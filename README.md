# TypeScript Learning Log

このリポジトリは、TypeScript学習の記録を日ごとにまとめたものだ。
インターン参加やチーム開発に向けて、基礎から実装までを段階的に学んでいく。

## 目的

- TypeScriptというトレンドの言語を触り、モダンプログラミングへの理解を深める
- 実行環境の構築から運用まで一通り経験する
- 学習過程を公開し、継続的に改善する

## 100日間の学習フェーズ
- 第1期：TypeScript基礎 & 開発環境 (Day 1 - 25)

    JavaScriptの基礎を抑えつつ、TypeScriptの基本構文と厳格な型システムに慣れる。

- 第2期：フレームワーク（React/Next.js）と実践的型定義 (Day 26 - 50)

    実戦で必須となるReactとNext.jsを学び、コンポーネント設計における型定義をマスターする。

- 第3期：メインプロダクト開発 (Day 51 - 85)

    目標プロダクト： 「AI機能付き・高機能タスク管理アプリケーション（Trelloクローン）」

    認証、データベース連携、ドラッグ＆ドロップ操作などを実装。

- 第4期：テスト・リファクタリング・面接対策 (Day 86 - 100)

    Jestによるテスト導入、GitHubでのポートフォリオ公開、想定問答の準備。

## マイルストーン
- Day 50： React/Next.jsを使って、API連携（天気予報や簡易メモ）ができるアプリを一つ公開できている。
- Day 100： 自作プロダクトをURL（Vercel等）で公開し、そのコードの型定義のこだわりを論理的に説明できる。
